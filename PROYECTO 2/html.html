HTML

HTML, que significa Lenguaje de Marcado de Hipertexto, es el lenguaje de marcado estándar utilizado para crear páginas web. Es la base de la mayoría de los sitios web y se utiliza para estructurar el contenido en Internet. HTML trabaja en conjunto con otras tecnologías como CSS (Hojas de Estilo en Cascada) y JavaScript para crear páginas web interactivas y visualmente atractivas.

HTML utiliza un sistema de etiquetas para estructurar el contenido en una página web. Las etiquetas están encerradas entre corchetes angulares (<>) y vienen en pares: una etiqueta de apertura y una etiqueta de cierre.
