CSS

CSS, o Hojas de Estilo en Cascada, es un lenguaje de hoja de estilo utilizado para describir la presentación y el formato de un documento escrito en HTML (Lenguaje de Marcado de Hipertexto). CSS permite a los desarrolladores web controlar el diseño, los colores, las fuentes y otros aspectos visuales de una página web, lo que lo convierte en una tecnología esencial para el diseño web.
